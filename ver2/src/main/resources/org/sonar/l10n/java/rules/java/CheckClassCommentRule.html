This rule detects invalid class comment

Noncompliant Code Example

class YourClassName{
	//TO DO
}

Compliant Solution

/**
 * Description for class MyClass_01
 * 
 * @author tathienphuoc
 * @see MyClass_01
 * @since J2EE 1.6
 */
class YourClassName{
	//TO DO
}

Note: Only description section and content of @author are required.
