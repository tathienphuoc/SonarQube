Floating Point Values

Floating-point numbers do not compare as equality (==). Floating-point numbers that should be equal are not always equal due to rounding problems. Such rounding problems may occur during computations.

Noncompliant Code Example

private void resetAngle(){
    double angle = 0.0;
    ...
    if (angle == 45.5) {
        setUpTime();
    }
}

Compliant Solution

private void calculateAngle(){
    Double angle = new Double(0.0);
    ...
    if (0 == angle.compareTo(new Double(45.5)) {
        setUpTime();
    }
}
