File Comment

Noncompliant Code Example

//Your imports
//Your class comments
public class YourClassName {
	//TO DO
}

Compliant Solution

/*=========================================================
*Copyright(c) 2022 CyberLogitec
*@FileName : YourClassName.java          
*@FileTitle : YourClassName           
*Open Issues : Your issues
*Change history : Your change history
*@LastModifyDate : 2022-08-11
*@LastModifier : Last modifier
*@LastVersion : 1.0
=========================================================*/
//Your imports
//Your class comments
public class YourClassName {
	//TO DO
}

Note

  • Only content of @FileName and @FileTitle are required.
  • Use at most one [Space] on each side of ':'.
  • The date format must use the same symbol (./-[Space]) between Year-Month and Month-Day. Ex: YYYY-MM-DD; YYYY.MM.DD.
